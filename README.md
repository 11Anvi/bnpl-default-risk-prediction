# BNPL Default Risk Prediction

Predicting customer default risk in Buy Now, Pay Later (BNPL) services using machine learning.

## Overview
Final M.Sc. Data Science capstone project (VIT). Built and compared Logistic Regression
and XGBoost classification models on a synthetic dataset of 3,200 customers to predict
default risk, with Logistic Regression outperforming across all evaluation metrics
(accuracy, ROC-AUC, recall).

## Approach
- Exploratory Data Analysis (EDA) and feature engineering
- Model building: Logistic Regression vs. XGBoost
- Evaluation: accuracy, ROC-AUC, recall, precision
- Business interpretation of results for decision-making

## Tools
Python, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib/Seaborn

- `defaultriskprediction.ipynb` — full notebook with EDA, modelling, and evaluation
